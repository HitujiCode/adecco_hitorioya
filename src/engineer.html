IT エンジニア科の・目指せる職業・取得可能資格・学習内容をご紹介します。

目指せる職業

  • インフラエンジニア

    活かせる資格・スキル
    • 基本情報技術者試験
    • Linux
    • CCNA
    • AWS

    インフラエンジニアは、システムやネットワークの設計、構築、保守、最適化を担当し、セキュリティ確保やドキュメンテーションも行います。

  • 社内SE

    活かせる資格・スキル
    • 基本情報技術者試験
    • ITパスポート
    • CCNA
    • AWS

    社内SEは、従業員のコンピュータやネットワークのトラブルシューティング、ソフトウェアのインストールとアップグレード、セキュリティ対策の実施などを行います。

  • クラウドエンジニア

    活かせる資格・スキル
    • 基本情報技術者試験
    • CCNA
    • AWS

    クラウドエンジニアは、クラウド環境上で動作するサーバーの設計・構築・運用・保守を担当。クラウドサービスの導入や移行、コスト管理、利用者のサポートも行います。

  • テクニカルサポート・運用保守

    活かせる資格・スキル
    • 基本情報技術者試験
    • ITパスポート

    テクニカルサポートは、顧客やユーザーからの技術的な問題や不具合に対応する役割です。トラブルシューティング、ソフトウェアやハードウェアの設定、操作方法の説明などを行います。

  • Linux系資格（LPIC・LinuC）

  • AWS認定資格

  • CCNA

  • ITパスポート

  • 基本情報技術者試験

学習内容

  1. 0

    コンピューターに関する基礎知識

    合計：２３時間

    ITにかかわる従事者として基本となる、コンピューターシステムの基礎、OS、データベース、インフラネットワーク、運用やインターネットなどコンピューター基礎を総括的に学びます。

  2. 0

    セキュリティに関する基礎知識

    合計：６時間

    エンジニアとして身に着けておくべき情報セキュリティスキルを学びます。攻撃の種類、攻撃への対応など情報セキュリティをエンジニアの立場で理解します。

  3. 0

    プログラミングに関する知識

    合計：３４ 時間

    基礎から応用までプログラミンング全般を学びます。JavaやPythonを効果的に学び実践での活用の基盤を作ります。Java SE Bronze の受験も可能な知識に到達します。

  4. 0

    インフラネットワークエンジニアの活用実践

    合計：４２ 時間

    Linux、CCNAなどのインフラエンジニアに必要な知識を学び、AWSなどのクラウドプラットフォームを理解し、実践で活用できる基盤を整えます。資格取得を目指せるプログラムとなっております。

  5. 0

    データサイエンス

    合計：５ 時間

    ビジネスデータ分析に必要な統計のスキルを身につけます。理論の学習はもとより、Excelを使って実際にデータ分析を行うことで、データサイエンスの基礎を理解します。

  6. 0

    全てのITエンジニアに必要な問題解決力

    合計：４時間

    社内やITの問題を定義し、解決の方向へ課題を論理的に整理する。真の課題を明確にして改善策を提案するための基本となる思考法を学びます。

  7. 0

    Digitalリテラシーを高める新技術の知識

    合計：１８時間

    課題解決の生産性向上を目的に、全てのビジネスパーソンに必要な新技術を効率よく効果的に学び、新技術を利活用する力を備えます。

    【学習内容】
    • AI
    • ビッグデータ
    • ロボット
    • クラウド
    • IoT
    • メタバース

  8. 0

    ITプロジェクトマネジメントに必要なコミュニケーションスキル

    合計：１８ 時間

    プロジェクトマネジメントスキルとして重視されるスキルの一つとしてコミュニケーション能力があります。
    聴く、そして適切な自己主張を主に復習し、業務に就いた時に即戦力として社内ユーザーやベンダーとのミーティングの場で円滑に進められるようにスキルを身に着けます。
