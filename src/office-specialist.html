オフィススペシャリスト科の・目指せる職業・取得可能資格・学習内容をご紹介します。

目指せる職業

  • 一般事務

    活かせる資格・スキル
    • MOS
    • ITパスポート
    • コミュニケーション能力

    一般事務とは、企業や団体で行われる一般的な事務作業を担当する職種です。 業務内容として、書類作成やデータ入力、電話対応、備品管理等があります。 企業や組織の運営を支える重要な役割を担っています。

  • 営業事務

    活かせる資格・スキル
    • MOS
    • ITパスポート
    • コミュニケーション能力

    営業事務とは、営業部門や営業職のサポートを行う職種です。業務内容として、書類作成や顧客管理、商品管理、データ集計等があります。
    営業事務は、営業部門や営業職の人たちが会社の商品やサービスをより多く売ることができるようバックアップ・サポートする職種となります。

  • 秘書

    活かせる資格・スキル
    • MOS
    • ITパスポート
    • 秘書検定

    秘書とは、企業の社長や役員などの要職に就く上司をサポートする職種です。業務内容として、電話・メール対応やスケジュール管理・調整、資料作成等のサポートを担う業務となります。上司が本業に集中できるよう裏で支える役割を担っています。

  • コールセンター

    活かせる資格・スキル
    • MOS
    • ITパスポート
    • コミュニケーション能力

    コールセンターとは、電話を主な手段として顧客からのお問い合わせやサービスの提供を行う職種です。業務内容として、インバウンド・アウトバウンド等があります。コールセンターでは、コミュニケーションスキル、MOSなどが働く上で有用なスキルとなってきます。

目指せる資格

  • 秘書検定

  • MOS

学習内容

  1. 0

    Officeアプリケーション活用講座

    合計：４ 時間

    MOSの学習の前に、基本スキルを復習します。 効果的に講座受講から資格試験の合格を目指します。

    【学習内容】
    • Word／Excelの基礎
    • Excel応用と関数

  2. 0

    テキスト MOS（Word365/Excel365）

    合計：３０時間

    資格取得に必要なスキルとタスクが網羅されたMOS試験対策専用教材で、必要なスキルを身に着けます。

    【学習内容】
    • Word365
    • Excel365

  3. 0

    業務で使うExcel講座（基礎・応用）

    合計：２．５時間

    Excelで日常行っている定型業務の自動化を図るために、記録機能を利用したExcelマクロの作成方法と利用方法を学習します。また、Excelのマクロ機能で自動作成されたプログラム（VBA）の見方、記録機能で作成したマクロを汎用性のある処理に改良するためのポイントについて学習します。

  4. 0

    Digitalリテラシーを高める新技術の知識

    合計：１８時間

    課題解決の生産性向上を目的に、全てのビジネスパーソンに必要な新技術を効率よく効果的に学び、新技術を利活用する力を備えます。

    【学習内容】
    • AI
    • ビッグデータ
    • ロボット
    • クラウド
    • IoT
    • メタバース

  5. 0

    ITパスポート試験対策講座

    合計：３７時間

    ITパスポートの試験内容の学習により、ITの基礎知識が身に付きます。日常のオフィスでの業務において、ITやDigitalの知識を身に着けることにより、より効率的に業務を進める礎になります。

    【学習内容】
    • 企業法務
    • 経営戦略
    • システム戦略・開発技術
    • プロジェクトマネジメント・サービスマネジメント
    • コンピューター基礎理論・コンピューターシステム

  6. 0

    問題解決力

    合計：４時間

    社内の問題を定義し、解決の方向性に向けるためにはその基礎となる思考法がしっかり身についていることが重要です。課題を論理的に整理していくだけでなく、真の課題を明確にし改善策を提案するために基本となる思考法を学びます。

    【学習内容】
    • カスタマーセントリシティ
    • ロジカルシンキング
    • デザインシンキング

  7. 0

    ビジネス現場で必要なコミュニケーションスキル/社会人としての基礎的リテラシー

    合計：４１時間

    ビジネスの現場では、さまざまなスキルが必要です。これらのスキルは、仕事の効率性を高めたり、他者と協力して仕事を進めたりするために役立ちます。 また、コミュニケーション能力が非常に重要です。
    傾聴力や適切な自己主張を主を復習し社内や顧客とのミーティングなど、業務に即戦力として円滑に進めることができるようになります。

    【学習内容】
    • アサーティブコミュニケーション
    • ネゴシエーション
    • 説得力
    • プレゼンテーション
