Contents

医療・介護科の・目指せる職業・取得可能資格・学習内容をご紹介します。

目指せる職業

  • 薬局等での医薬品販売

    活かせる資格・スキル
    • 登録販売者資格
    • コミュニケーション能力

    薬局やドラッグストア・一部のスーパーマーケットなどで、処方箋を必要としない市販薬や健康補助食品を一般の顧客に販売する職種です。顧客の健康と安全を守りながら、適切な医薬品の提供とサポートを行います。

  • 介護事務管理士

    活かせる資格・スキル
    • 介護事務管理士
    • コミュニケーション能力
    • 情報管理能力

    介護事務管理士は、介護施設や在宅介護支援事業所で働く事務担当者です。彼らは日常の事務業務を管理し、介護費用の請求や利用者のケアプランの立案も行います。
    患者や利用者のサポートを通じて、施設の円滑な運営と利用者のニーズへの適切な対応を支援します。

  • 受付スタッフ

    活かせる資格・スキル
    • 基本的なPCスキル
    • Word、Excel、Powerpoint等

    患者や顧客との直接対応が主な仕事です。処方箋の受付や調剤、保険や支払い手続きの案内、在庫管理、電話対応、薬剤師や医師との連絡調整、清算業務などを担当します。丁寧なコミュニケーション能力が重要であり、スムーズな薬局等の運営に貢献します。

  • 事務スタッフ

    活かせる資格・スキル
    • 基本的なPCスキル
    • Word、Excel、Powerpoint等

    事務スタッフとして、患者の診療記録や施設の運営データの管理、統計データの集計など、様々な業務を担当したり、患者や利用者からの問い合わせや相談に対応し、受付業務や電話応対を行います。患者や利用者との円滑なコミュニケーション能力が重要であり、要望や不安を丁寧に聞き、適切な対応や案内を行います。

目指せる資格

  • 登録販売者資格

  • 介護事務管理士

  • 医療事務

  • 販売士

  • MOS（Microsoft Office Specialist）

学習内容

  1. 0

    コース①　登録販売者資格取得講座

    合計：７０ 時間

    登録販売者とは、ドラッグストアや薬局などで一般用医薬品の販売ができる医薬品販売の専門資格です。その資格取得を目指すための対策講座です。

    【学習内容】
    • 医薬品に共通する特性と基本的な知識
    • 人体の働きと医薬品
    • 主な医薬品とその作用
    • 薬事関係法規・制度
    • 医薬品の適正使用・安全対策

    コース②　介護事務管理士取得講座

    合計：５０時間

    介護事務管理士の講座では、介護施設での業務を支援する役割を学びます。介護費用請求やケアプランの立案などを通じて、居宅介護支援事業所のケアマネジャーをサポートします。また、実践的な介護知識やストレスマネジメント、クレーム対応なども学びます。

  2. 0

    働く上でのコミュニケーション

    合計：２時間

    老若男女様々な方を対応しなければいけない医薬業界。どんな方でも主訴を的確に理解し、受けたいサービスを把握したうえで、丁寧に説明し、伝える力が必要です。主には、コミュニケーション能力として、聴く力、伝える力を復習し、業務に就いた時に即戦力として顧客・患者さん・社内の職員などの関わる方との適切で円滑な対応スキルを身に着けます。

  3. 0

    様々な情報セキュリティ

    合計：２時間

    医薬業では機微情報を取り扱います。ほかの産業以上に注意が必要ですが、一般的な情報セキュリティを理解します。また、SNS等での情報配信や身近にある情報リスクについても学び社会人として身に着けておくべき情報セキュリティスキルを総括的に身に着けます。

    介護事務管理士の講座では、介護施設での業務を支援する役割を学びます。介護費用請求やケアプランの立案などを通じて、居宅介護支援事業所のケアマネジャーをサポートします。また、実践的な介護知識やストレスマネジメント、クレーム対応なども学びます。

  4. 0

    OA操作・Office アプリケーション活用

    合計：６時間

    医薬従事者は専用の端末を使うことが多いですが、医薬業界は接遇だけが仕事ではありません。裏方として、事務業務も発生します。日々の報告や進捗の可視化などあらゆる局面でデジタル化が進んでおり、その基本となるOfficeアプリケーションを活用できるようになります。

  5. 0

    知っておくべきDX基礎

    合計：１８時間

    Digitalによる変化をダイジェスト的に学びます。課題解決の生産性向上を目的に、全てのビジネスパーソンに必要な新技術を効率よく効果的に学び、新技術を利活用する力を備えます。

    【学習内容】
    • AI
    • ビッグデータ
    • ロボット
    • クラウド
    • IoT
    • メタバース

  6. 0

    社会人としての基礎的リテラシー

    合計：４５時間

    働く業界を問わず高めておきたい基礎的リテラシーを総括的に学びます。
    ビジネスの現場では様々なリスクへの対応としてコンプライアンス講座やSNSのリスク対応講座などから効率的な仕事の進め方、さらにはタイムマネジメントなどの自身の仕事の仕方、また、他者との仕事をすることが多い環境に即した説明力を強化するスキルなどを身に着けます。社会人として身に着けておきたい基礎的なスキルを振り返りながら補っていきます。
