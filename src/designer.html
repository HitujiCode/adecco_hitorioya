目指せる職業

  • Webデザイナー

    活かせる資格・スキル
    • ウェブデザイン技能検定3級

    Webデザイナーは、目的やコンセプトに合わせたWebサイトを設計する職種です。業務内容として、クライアントの意向をヒアリングしながら、デザインの原則やHTMLやCSS、Javaなどを用いて、視覚的に魅力的で使いやすいWebの作成を行います。

  • Webオペレーター

    活かせる資格・スキル
    • PCスキル
    • Web検定
    • ウェブデザイン技能検定3級

    Webオペレーターは、Webサイトの管理と運用を行う職種です。業務内容として、新しいコンテンツの追加や修正、顧客からの問い合わせへの返信、Webサイトの動作確認やトラブル解決などを担当します。
    初心者にとっては、Webサイトの使い方や基本的なコンピュータスキルが求められますが、経験を積むことで成長できるポジションです。

  • Webマーケター

    活かせる資格・スキル
    • GAIQ
    • SEO資格
    • Google Analytics等の知識

    Webマーケターは、Webサイトやソーシャルメディアを活用して商品やサービスの宣伝や販促を行う職種です。業務内容として、デジタル広告戦略の立案・実行やSEO、SNSキャンペーンの管理、顧客行動の分析などを担当し、オンラインでのビジネス成長を促します。

  • UX・UIデザイナー

    活かせる資格・スキル
    • ウェブデザイン技能検定3級
    • HTML5プロフェッショナル認定試験

    UXデザイナーは、ユーザが製品やサービスをどのように体験するかを設計し、UIデザイナーは、その設計を実装し、ユーザが直感的に操作できるインターフェースを作成する職種です。業務内容として、設計の方向性の決定や、目標値の設定に合わせた導線の確認、テスト運用などがあります。

学習内容

  1. 0

    PC基礎知識とネットリテラシー

    合計：３ 時間

    はじめにWebで仕事をしていくための必須となる、ファイルの操作方法やエディタ等の基礎のPC知識を学びます。それらを踏まえたうえで、インターネットを活用する上でのリスクへの備えから、最新情報を踏まえ、効果的な活用方法までを学び、インターネットリテラシーを高めます。

  2. 0

    Webの基礎知識

    合計：４１時間

    Webサイトがどのようなシステムで動いているのかを理解することで、制作をする上での基本が身に付きます。また、Webサイト構築の基本を学び、Webが社会と密接につながっていることを理解し今後の仕事の礎にします。

  3. 0

    Webサイト制作の基礎と演習

    合計：４０時間

    Webサイトを制作する上で、まず具体的なスキルを身につける前に全体像を仕組みから理解しWeb制作の基礎を学びます。次にWeb制作で必須となる、HTLMとCSSを学びます。Webページを作成するための言語と作られた文書に色や大きさなどの見た目をデザインする言語を学びます。いずれもWebサイトの見た目を整える上で欠かせないプログラミング言語です。基礎を学び、手を動かしながら学びます。

  4. 0

    Figma基礎・実践

    合計：２１時間

    ブラウザー上で簡単にデザインができるFigmaの画面構成から、基本操作、そして演習とプロトタイピングを制作します。

    【学習手順】

    Figmaの概要→特徴と操作性→効率的なデザイン制作

  5. 0

    UX/UI基礎

    合計：５時間

    UX/UIの全体像、視覚デザインやインターフェイスデザインのポイントおよびUXの戦略的手法のポイントを理解します。UXデザイナーに求められるスキルとして、UXデザインのプロセス、UXに影響する技術的要因、現実的なアクセシビリティの配慮、UXに関連する技術の進化と社会的課題およびUXデザインとビジネスなどの理解を深めます。ビジネスに活かすUXデザインの手法により効果的な訴求方法を学びます。

  6. 0

    Webマーケター基礎

    合計：４０時間

    Webマーケティングやインターネット広告マーケティングの基礎から効果検証方法を学びます。 また、セールスライティングや写真撮影の演習を交えて効果的な表現を学びます。

    【学習手順】

    Webマーケットとは→インターネット広告の基礎→Webマーケティングの基礎

    SNSマーケティングとは→アクセス解析の方法→セールスライティング

    写真撮影の基礎と演習
