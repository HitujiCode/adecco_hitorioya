目指せる職業

  • プログラマー

    活かせる資格・スキル
    • ITパスポート
    • OCP
    • CCNA

    コンピューター、システム、プログラミングなどの知識を使って、多くの人々に役立つシステムを作り上げるやりがいのあるお仕事です。IT化やDX化を推進する現代では欠かせない存在です。

  • デバッガー

    活かせる資格・スキル
    • 論理的な思考力やユーザー目線で考える想像力

    ゲームやアプリなどのソフトウェアのプログラムのミス（バグ）を見つけて、原因を分析し、正常に動作するように修正するお仕事です。制作を支える重要な役割を担います。

  • テスター

    活かせる資格・スキル
    • 基礎的OAスキル
    • 集中力
    • IVEC

    システムやアプリケーション、IT製品が正常に動作するかをテスト・評価するエンジニアのお仕事です。デジタルライフを支えるためのとても大切な存在です。

  • ソフトウエア・ゲーム開発エンジニア

    活かせる資格・スキル
    • コミュニケーション力
    • ITパスポート
    • 応用情報技術者試験

    ゲームの企画・制作に携わり、プログラミング言語を使用し、ゲームの開発を行います。チームで作業を進めることも多く、クリエイティブなエンジニアのお仕事です。

学習内容

  1. 0

    IT基礎理論

    合計：２０時間

    ITエンジニアに共通の基礎知識として、初心者の方は全般を捉えることを目的に、経験の有る方は復習として活用し、知識を学んでいきます。（コンピューター、オペレーティングシステム、ネットワーク、システム開発、Web、運用管理）

  2. 0

    Javaプログラミング

    合計：３７時間

    Java言語を用いたプログラムの作成方法、基本文法、オブジェクト指向の基本となる考え方やJavaでの実装方法についてが学習します。

  3. 0

    Pythonプログラミング

    合計：４２ 時間

    Pythonでプログラムを作成するために必要な基礎知識を実習を通して修得します。オブジェクト指向プログラミングの基礎（インスタンスの作成と利用、継承、例外処理など）をPythonで実現する方法を学びます。

  4. 0

    データベース（MYSQL）

    合計：２３ 時間

    データベースの設計、構築、運用管理を実施するために必要な、プロダクトに依存しないデータベースの要点を理解していただきます。（MYSQL/リレーショナルデータベース/クライアントサーバーモデル）

  5. 0

    AI・IoT概論

    合計：１７ 時間

    機械学習やディープラーニングなどのAIは、何の役に立つのか、どのように使うのかについて学習し、活用のポイントを整理します。
    また、センサー、クラウド、ネットワーク、通信プロトコル、機械学習などのIoTの概要を学習します。

  6. 0

    ITプロジェクトマネジメントに必要なコミュニケーションスキル

    合計：９時間

    ITエンジニアのとして重視されるスキルの一つであるコミュニケーション能力開発のために、聴く、適切な自己主張について復習した上で、報告書、交渉術、ロジカルライティング、プレゼンテーションについても学びます。

  7. 0

    全てのITエンジニアに必要な問題解決力

    合計：４時間

    社内やITの問題を定義し、解決するための思考法を身に付けていただきます。課題を論理的に整理した上で、真の課題を明確にし改善策を提案するために基本となる思考法として、カスタマーセントリシティ、ロジカルシンキング、デザインシンキングを学習します。
