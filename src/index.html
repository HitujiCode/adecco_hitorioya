当事業のポイント

Point

  1. Point0

    すき間時間を活かして学べる
    eラーニング

    家事の合間や移動中などの貴重なすき間時間を活かして学ぶことができます。パソコンやスマホで受講可能なので、いつでもどこでも自分のペースで学習を進めることができます。

  2. Point0

    専任キャリアアドバイザーと
    メンターにいつでも相談

    専任キャリアアドバイザーが就職支援、メンターが学習支援に特化し、あなたのそばでサポートします。
    いつでも相談できるので、安心してキャリアアップいただけます。

  3. Point0

    専用求人で今より一歩上の
    理想のキャリアを実現

    受講者専用の求人を用意し、ありたい姿や価値観、スキルや経験、希望条件などを参考にした専用求人を公開します。また、企業説明会なども実施し、気になる企業を見つける手助けも行います。

  4. Point0

    育児の悩みも育児支援
    アドバイザーに相談できる

    育児に関する悩みや不安など、専門の育児支援アドバイザーにご相談いただけます。バーチャル空間を利用することで、お好きな場所からアドバイザーとリアルタイムで会話することができます。

参加の流れ

Flow

  • 0

    簡単入力で日程調整

    お申し込みフォームから必要な情報を入力し、面談日程をご選択ください。

  • 0

    事業説明と面談

    予約いただいた面談日にて、まずは事業の説明をいたします。面談時には、あなたの希望や適性、学習計画などを確認し、その後、キャリアや資格の習得に向けて、科目の訓練を開始いたします。

  • 0

    eラーニングでスキル取得

    一人ひとりの生活スタイルに合わせて、一緒に学習計画を立てます。すき間時間を学習時間にあてて、eラーニングでスキルアップを目指しましょう。受講中にわからないことがあれば、メンターが丁寧にサポートします。

  • 0

    就職に向けたサポート

    訓練と並行し、カウンセリングや個別相談で就職を支援。履歴書作りや面接対策もアドバイス。希望に合った企業を紹介しマッチングを支援

受講条件

conditions

下記３つの条件を全て満たす方が対象です。

  • ①ひとり親の方で仕事と家庭の両立のために、eラーニング等による訓練受講を希望する方。
  • ②都内に在住または都内で在勤している方。
  • ③令和５年度・令和６年度に「ひとり親向けオンライン訓練・就活支援事業」や「類似事業」の受講が決定していない方。
    ※類似事業は「成長産業分野キャリア形成支援事業」「女性向けキャリアチェンジ支援事業」「女性ITエンジニア育成事業（令和５年度のみ対象）
