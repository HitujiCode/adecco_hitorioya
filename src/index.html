当事業のポイント

Point

  1. Point0

    すき間時間を活かして学べる
    eラーニング

    家事の合間や移動中などの貴重なすき間時間を活かして学ぶことができます。パソコンやスマホで受講可能なので、いつでもどこでも自分のペースで学習を進めることができます。

  2. Point0

    専任キャリアアドバイザーと
    メンターにいつでも相談

    専任キャリアアドバイザーが就職支援、メンターが学習支援に特化し、あなたのそばでサポートします。
    いつでも相談できるので、安心してキャリアアップいただけます。

  3. Point0

    専用求人で今より一歩上の
    理想のキャリアを実現

    受講者専用の求人を用意し、ありたい姿や価値観、スキルや経験、希望条件などを参考にした専用求人を公開します。また、企業説明会なども実施し、気になる企業を見つける手助けも行います。

  4. Point0

    育児の悩みも育児支援
    アドバイザーに相談できる

    育児に関する悩みや不安など、専門の育児支援アドバイザーにご相談いただけます。バーチャル空間を利用することで、お好きな場所からアドバイザーとリアルタイムで会話することができます。

受講条件

Condtions

下記３つの条件を全て満たす方が対象です。

  • ①ひとり親の方で仕事と家庭の両立のために、eラーニング等による訓練受講を希望する方。
  • ②都内に在中または都内で在勤している方。
  • ③令和５年度、令和６年度に「成長産業分野キャリア形成支援事業」や「ひとり親向けオンライン訓練・就職支援事業」、「女性向けキャリアチェンジ支援事業」等の類似訓練の受講が決定していない方。
